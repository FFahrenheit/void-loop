PLACAS DE DESARROLLO

Arduino UNO R3

El UNO es la mejor placa para empezar con la electrónica y el desarrollo de software. Si esta es tu primera experiencia con la plataforma, Arduino UNO es para ti.

Arduino MEGA R3

El MEGA 2560 está diseñado para proyectos más complejos. Con 54 pines de I/O digital y 16 análogos te permite hacer projectos más complejos y largos, como impresoras 3D.

Arduino Nano R3

El Arduino Nano es una placa compacta similar a Arduino UNO. Al ser de tamaño reducido, permite hacer prototipos má discretos.
